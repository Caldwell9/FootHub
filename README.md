# StepStyle - MERN E-Commerce Platform

A full-stack e-commerce web application for a shoe brand built with MongoDB, Express, React, and Node.js.

## 🎯 Project Overview

**StepStyle** is a modern, responsive e-commerce platform with the following features:
- User authentication and authorization
- Product catalog with categories
- Shopping cart functionality
- Order checkout and placement
- Admin panel for product and order management
- Rating and review system
- Responsive design (Mobile & Desktop)
- Black & White theme with Red accents

---

## 📋 Tech Stack

### Frontend
- **React 18** - UI library with functional components and hooks
- **React Router v6** - Navigation and routing
- **Axios** - HTTP client
- **CSS3** - Custom styling with modern design

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB ODM
- **JWT** - Authentication
- **bcryptjs** - Password hashing

### Tools
- **Vite** - Frontend build tool
- **Nodemon** - Development server
- **npm/yarn** - Package managers
